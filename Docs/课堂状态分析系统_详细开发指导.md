# 课堂状态分析系统：详细开发指导文件

> 项目主题：基于多人人脸检测与表情识别的课堂状态分析系统  
> 推荐实现：Python + OpenCV + Streamlit + 表情识别模型  
> 开发方式：分步迭代，先完成基础版，再逐步加入加分功能和延展任务

---

## 0. 项目最终目标

本项目需要实现一个本地运行的课堂状态分析系统。系统能够接收图片、视频或摄像头画面，检测画面中的多个人脸，对每个人脸进行表情识别，统计不同表情的人数与比例，并根据统计结果给出课堂状态判断。同时，系统需要提供本地可视化界面，并保存每次检测记录。

最终系统至少应具备以下能力：

1. 上传一张多人课堂图片。
2. 自动检测图片中的所有人脸。
3. 在图片中画出人脸框。
4. 对每个人脸识别表情。
5. 显示检测人数、各类表情人数、各类表情比例。
6. 输出课堂状态，例如“状态良好”“状态平稳”“需要关注”。
7. 保存识别记录到 `records.csv`。
8. 使用 Streamlit 或 Gradio 提供本地可视化界面。
9. 加分方向包括视频分析、摄像头实时识别、统计图表、结果图片导出、多图批量分析、时序预警等。

## 1. 推荐技术路线

### 1.1 基础版技术选择

| 模块 | 推荐方案 | 理由 |
|---|---|---|
| 可视化界面 | Streamlit | 代码简单，适合课程大作业演示 |
| 人脸检测 | OpenCV Haar Cascade | 安装简单，容易调试 |
| 表情识别 | 课堂训练模型 / FER2013 模型 / DeepFace | 可根据已有资源选择 |
| 统计分析 | Python Counter + Pandas | 实现简单，便于保存记录 |
| 结果可视化 | Streamlit 表格 + Matplotlib | 直观展示统计结果 |
| 数据记录 | CSV | 实现难度低，符合要求 |

### 1.2 推荐开发顺序

不要一开始就做完整系统。建议按照以下顺序迭代：

1. 先实现图片上传。
2. 再实现多人脸检测。
3. 再实现单张人脸表情识别。
4. 再把多人脸检测和表情识别连接起来。
5. 再做统计与课堂状态判断。
6. 再做 Streamlit 界面。
7. 再做记录保存。
8. 最后做视频、摄像头、时序预警等加分项。

### 5.3 utils.py：图片格式转换工具

Streamlit 上传的文件需要转换成 OpenCV 可处理的图像。

```python
# utils.py

import cv2
import numpy as np
from PIL import Image


def uploaded_file_to_bgr(uploaded_file):
    """
    将 Streamlit 上传文件转换为 OpenCV BGR 图像。
    """
    image = Image.open(uploaded_file).convert("RGB")
    image_np = np.array(image)
    image_bgr = cv2.cvtColor(image_np, cv2.COLOR_RGB2BGR)
    return image_bgr


def bgr_to_rgb(image_bgr):
    """
    OpenCV BGR 图像转 RGB，用于 Streamlit 展示。
    """
    return cv2.cvtColor(image_bgr, cv2.COLOR_BGR2RGB)


def crop_face(image_bgr, box, margin=0.15):
    """
    根据人脸框裁剪人脸区域。
    box: (x, y, w, h)
    margin: 适当扩大人脸框，避免裁剪过紧。
    """
    h_img, w_img = image_bgr.shape[:2]
    x, y, w, h = box

    dx = int(w * margin)
    dy = int(h * margin)

    x1 = max(0, x - dx)
    y1 = max(0, y - dy)
    x2 = min(w_img, x + w + dx)
    y2 = min(h_img, y + h + dy)

    return image_bgr[y1:y2, x1:x2]
```

### 5.4 face_detector.py：OpenCV Haar 人脸检测

```python
# face_detector.py

import cv2


class FaceDetector:
    """
    基于 OpenCV Haar Cascade 的人脸检测器。
    优点：安装简单，适合大作业基础版。
    缺点：复杂角度、遮挡、低清图像下可能漏检。
    """

    def __init__(self):
        cascade_path = cv2.data.haarcascades + "haarcascade_frontalface_default.xml"
        self.detector = cv2.CascadeClassifier(cascade_path)

        if self.detector.empty():
            raise RuntimeError("Haar Cascade 加载失败，请检查 OpenCV 安装。")

    def detect(self, image_bgr):
        """
        输入 BGR 图像，返回人脸框列表。
        返回格式：[{"box": (x, y, w, h)}, ...]
        """
        gray = cv2.cvtColor(image_bgr, cv2.COLOR_BGR2GRAY)

        faces = self.detector.detectMultiScale(
            gray,
            scaleFactor=1.1,
            minNeighbors=5,
            minSize=(40, 40)
        )

        results = []
        for (x, y, w, h) in faces:
            results.append({
                "box": (int(x), int(y), int(w), int(h))
            })

        return results

    def draw_faces(self, image_bgr, detections):
        """
        在图像上绘制人脸框。
        """
        output = image_bgr.copy()

        for item in detections:
            x, y, w, h = item["box"]
            cv2.rectangle(output, (x, y), (x + w, y + h), (0, 255, 0), 2)

        return output
```

### 6.4 expression_recognizer.py：推荐写法

下面的写法支持两种模式：

- 如果存在本地 Keras 模型，则使用本地模型。
- 如果没有模型，则尝试使用 DeepFace。
- 如果都不可用，会给出明确错误提示。

```python
# expression_recognizer.py

import cv2
import numpy as np
from pathlib import Path
from config import EMOTION_LABELS


class ExpressionRecognizer:
    """
    表情识别模块。
    优先使用本地 Keras 模型；
    如果没有本地模型，可以改为调用 DeepFace。
    """

    def __init__(self, model_path=None, backend="keras"):
        self.backend = backend
        self.model = None

        if backend == "keras":
            self._load_keras_model(model_path)
        elif backend == "deepface":
            self._load_deepface()
        else:
            raise ValueError("backend 只能是 'keras' 或 'deepface'")

    def _load_keras_model(self, model_path):
        if model_path is None:
            raise ValueError("使用 keras 后端时必须提供 model_path")

        model_path = Path(model_path)
        if not model_path.exists():
            raise FileNotFoundError(f"模型文件不存在：{model_path}")

        from tensorflow.keras.models import load_model
        self.model = load_model(model_path)

    def _load_deepface(self):
        try:
            from deepface import DeepFace
            self.DeepFace = DeepFace
        except Exception as e:
            raise RuntimeError("DeepFace 加载失败，请检查是否安装 deepface 和 tf-keras") from e

    def predict(self, face_bgr):
        """
        输入裁剪后的人脸图像，返回：
        {
            "label": "Happy",
            "confidence": 0.92
        }
        """
        if self.backend == "keras":
            return self._predict_with_keras(face_bgr)

        if self.backend == "deepface":
            return self._predict_with_deepface(face_bgr)

    def _predict_with_keras(self, face_bgr):
        """
        假设模型输入为 48x48 灰度图。
        如果你的模型输入尺寸不同，需要修改这里。
        """
        gray = cv2.cvtColor(face_bgr, cv2.COLOR_BGR2GRAY)
        resized = cv2.resize(gray, (48, 48))
        normalized = resized.astype("float32") / 255.0

        x = np.expand_dims(normalized, axis=-1)
        x = np.expand_dims(x, axis=0)

        preds = self.model.predict(x, verbose=0)[0]
        idx = int(np.argmax(preds))
        confidence = float(preds[idx])

        label = EMOTION_LABELS[idx] if idx < len(EMOTION_LABELS) else str(idx)

        return {
            "label": label,
            "confidence": confidence
        }

    def _predict_with_deepface(self, face_bgr):
        """
        使用 DeepFace 分析表情。
        注意：DeepFace 返回的表情名称通常是小写英文，需要映射到项目标签。
        """
        face_rgb = cv2.cvtColor(face_bgr, cv2.COLOR_BGR2RGB)

        result = self.DeepFace.analyze(
            img_path=face_rgb,
            actions=["emotion"],
            enforce_detection=False
        )

        if isinstance(result, list):
            result = result[0]

        emotion_scores = result.get("emotion", {})
        dominant = result.get("dominant_emotion", "neutral")

        mapping = {
            "happy": "Happy",
            "neutral": "Neutral",
            "sad": "Sad",
            "angry": "Angry",
            "surprise": "Surprise",
            "fear": "Fear",
            "disgust": "Disgust",
        }

        label = mapping.get(dominant.lower(), "Neutral")
        confidence = float(emotion_scores.get(dominant, 0.0)) / 100.0

        return {
            "label": label,
            "confidence": confidence
        }
```

### 7.2 修改 face_detector.py

把原来的 `draw_faces` 改成更完整的 `draw_results`。

```python
# face_detector.py

import cv2


class FaceDetector:
    def __init__(self):
        cascade_path = cv2.data.haarcascades + "haarcascade_frontalface_default.xml"
        self.detector = cv2.CascadeClassifier(cascade_path)

        if self.detector.empty():
            raise RuntimeError("Haar Cascade 加载失败，请检查 OpenCV 安装。")

    def detect(self, image_bgr):
        gray = cv2.cvtColor(image_bgr, cv2.COLOR_BGR2GRAY)

        faces = self.detector.detectMultiScale(
            gray,
            scaleFactor=1.1,
            minNeighbors=5,
            minSize=(40, 40)
        )

        return [{"box": tuple(map(int, face))} for face in faces]

    def draw_results(self, image_bgr, detections):
        output = image_bgr.copy()

        for item in detections:
            x, y, w, h = item["box"]

            label = item.get("emotion", "Face")
            confidence = item.get("confidence", None)

            if confidence is not None:
                text = f"{label} {confidence:.2f}"
            else:
                text = label

            cv2.rectangle(output, (x, y), (x + w, y + h), (0, 255, 0), 2)

            text_y = y - 10 if y - 10 > 20 else y + 20
            cv2.putText(
                output,
                text,
                (x, text_y),
                cv2.FONT_HERSHEY_SIMPLEX,
                0.7,
                (0, 255, 0),
                2,
                cv2.LINE_AA
            )

        return output
```

# analyzer.py

from collections import Counter
from config import (
    EMOTION_LABELS,
    GOOD_THRESHOLD,
    LOW_THRESHOLD,
    SURPRISE_THRESHOLD,
    HIGH_NEUTRAL_THRESHOLD,

def summarize_expressions(detections):
    """
    detections: [
        {"box": ..., "emotion": "Happy", "confidence": 0.9},
        ...
    ]

    返回统计结果字典。
    """
    total = len(detections)
    counter = Counter()

    for item in detections:
        emotion = item.get("emotion", "Unknown")
        counter[emotion] += 1

    counts = {label: counter.get(label, 0) for label in EMOTION_LABELS}

    if total > 0:
        ratios = {label: counts[label] / total for label in EMOTION_LABELS}
        main_expression = max(counts, key=counts.get)
    else:
        ratios = {label: 0.0 for label in EMOTION_LABELS}
        main_expression = "None"

    status = judge_classroom_status(total, counts, ratios, main_expression)

    return {
        "total": total,
        "counts": counts,
        "ratios": ratios,
        "main_expression": main_expression,
        "status": status,
    }


def judge_classroom_status(total, counts, ratios, main_expression):
    """
    根据表情比例判断课堂状态。
    规则可以根据实际效果微调。
    """
    if total == 0:
        return "未检测到学生"

    positive_ratio = ratios.get("Happy", 0) + ratios.get("Neutral", 0)
    low_ratio = ratios.get("Sad", 0) + ratios.get("Angry", 0)
    surprise_ratio = ratios.get("Surprise", 0)
    neutral_ratio = ratios.get("Neutral", 0)

    if low_ratio >= LOW_THRESHOLD:
        return "课堂状态较低落或需要关注"

    if surprise_ratio >= SURPRISE_THRESHOLD:
        return "课堂注意力波动较大"

    if positive_ratio >= GOOD_THRESHOLD:
        return "课堂状态良好"

    if main_expression == "Neutral" or neutral_ratio >= HIGH_NEUTRAL_THRESHOLD:
        return "课堂状态平稳"

    return "课堂状态一般"
```

# recorder.py

from datetime import datetime
from pathlib import Path
import pandas as pd
from config import RECORD_FILE, EMOTION_LABELS


def build_record(image_name, summary, result_path=None):
    """
    构造单次图片检测记录。
    """
    record = {
        "time": datetime.now().strftime("%Y-%m-%d %H:%M:%S"),
        "image_name": image_name,
        "total_people": summary["total"],
        "main_expression": summary["main_expression"],
        "classroom_status": summary["status"],
        "result_path": result_path or "",
    }

    for label in EMOTION_LABELS:
        record[f"{label}_count"] = summary["counts"].get(label, 0)
        record[f"{label}_ratio"] = round(summary["ratios"].get(label, 0.0), 4)

    return record


def append_record(image_name, summary, result_path=None, file_path=RECORD_FILE):
    """
    追加保存一条检测记录。
    """
    file_path = Path(file_path)
    record = build_record(image_name, summary, result_path)
    df = pd.DataFrame([record])

    if file_path.exists():
        df.to_csv(file_path, mode="a", index=False, header=False, encoding="utf-8-sig")
    else:
        df.to_csv(file_path, mode="w", index=False, header=True, encoding="utf-8-sig")


def load_records(file_path=RECORD_FILE):
    """
    加载历史记录。
    """
    file_path = Path(file_path)

    if not file_path.exists():
        return pd.DataFrame()

    return pd.read_csv(file_path, encoding="utf-8-sig")

# video_processor.py

import cv2
import tempfile
import pandas as pd
from analyzer import summarize_expressions
from utils import crop_face


def save_uploaded_video(uploaded_video):
    """
    Streamlit 上传的视频无法直接被 cv2.VideoCapture 稳定读取，
    建议先保存成临时文件。
    """
    suffix = "." + uploaded_video.name.split(".")[-1]

    with tempfile.NamedTemporaryFile(delete=False, suffix=suffix) as tmp:
        tmp.write(uploaded_video.read())
        return tmp.name


def analyze_video(video_path, detector, recognizer, interval_sec=1):
    """
    按 interval_sec 秒抽一帧进行分析。
    返回：
    - frame_records: 每帧统计结果列表
    - total_summary: 全视频汇总结果
    """
    cap = cv2.VideoCapture(video_path)

    if not cap.isOpened():
        raise RuntimeError("视频打开失败")

    fps = cap.get(cv2.CAP_PROP_FPS)
    if fps <= 0:
        fps = 25

    frame_interval = int(fps * interval_sec)
    frame_index = 0
    sampled_index = 0

    frame_records = []
    all_detections = []

    while True:
        ret, frame_bgr = cap.read()
        if not ret:
            break

        if frame_index % frame_interval == 0:
            timestamp_sec = frame_index / fps

            detections = detector.detect(frame_bgr)

            for item in detections:
                face_img = crop_face(frame_bgr, item["box"])
                pred = recognizer.predict(face_img)
                item["emotion"] = pred["label"]
                item["confidence"] = pred["confidence"]

            summary = summarize_expressions(detections)

            record = {
                "frame_no": sampled_index,
                "video_time_sec": round(timestamp_sec, 2),
                "total_people": summary["total"],
                "main_expression": summary["main_expression"],
                "classroom_status": summary["status"],
            }

            for label, count in summary["counts"].items():
                record[f"{label}_count"] = count
                record[f"{label}_ratio"] = round(summary["ratios"][label], 4)

            frame_records.append(record)
            all_detections.extend(detections)

            sampled_index += 1

        frame_index += 1

    cap.release()

    total_summary = summarize_expressions(all_detections)
    return frame_records, total_summary


def records_to_dataframe(frame_records):
    return pd.DataFrame(frame_records)
```

# analyzer.py

import pandas as pd


def add_warning_levels(frame_df, window_size=5):
    """
    输入逐帧分析 DataFrame，输出带预警等级的 DataFrame。
    要求 frame_df 至少包含：
    - Happy_ratio
    - Neutral_ratio
    - Sad_ratio
    - Angry_ratio
    - Surprise_ratio
    - classroom_status
    """

    if frame_df.empty:
        return frame_df

    df = frame_df.copy()

    positive = df.get("Happy_ratio", 0) + df.get("Neutral_ratio", 0)
    low = df.get("Sad_ratio", 0) + df.get("Angry_ratio", 0)
    surprise = df.get("Surprise_ratio", 0)
    neutral = df.get("Neutral_ratio", 0)

    df["positive_ratio"] = positive
    df["low_ratio"] = low
    df["attention_wave_ratio"] = surprise

    df["window_positive_mean"] = df["positive_ratio"].rolling(
        window=window_size,
        min_periods=1
    ).mean()

    df["window_low_mean"] = df["low_ratio"].rolling(
        window=window_size,
        min_periods=1
    ).mean()

    df["window_neutral_mean"] = neutral.rolling(
        window=window_size,
        min_periods=1
    ).mean()

    df["warning_level"] = "None"

    good_streak = 0
    neutral_streak = 0
    low_streak = 0

    for idx in df.index:
        current_positive = df.loc[idx, "positive_ratio"]
        current_neutral = df.loc[idx, "Neutral_ratio"]
        current_low = df.loc[idx, "low_ratio"]
        current_surprise = df.loc[idx, "attention_wave_ratio"]

        if current_positive >= 0.70:
            good_streak += 1
        else:
            good_streak = 0

        if current_neutral >= 0.60:
            neutral_streak += 1
        else:
            neutral_streak = 0

        if current_low >= 0.40 or current_surprise >= 0.30:
            low_streak += 1
        else:
            low_streak = 0

        if low_streak >= 3:
            df.loc[idx, "warning_level"] = "Red"
        elif neutral_streak >= 5:
            df.loc[idx, "warning_level"] = "Yellow"
        elif good_streak >= 10:
            df.loc[idx, "warning_level"] = "Green"
        else:
            df.loc[idx, "warning_level"] = "None"

    return df

## 21. 常见问题与解决方案

### 21.1 Haar 检测不到人脸

可能原因：

- 人脸太小。
- 人脸侧脸角度过大。
- 图片模糊。
- 光照太暗。
- 参数过严。

解决：

```python
faces = detector.detectMultiScale(
    gray,
    scaleFactor=1.05,
    minNeighbors=3,
    minSize=(30, 30)
)
```

如果仍然效果不好，可以替换为 MediaPipe Face Detection 或 MTCNN。

---

### 21.2 表情识别结果不准

可能原因：

- 人脸裁剪太紧。
- 输入尺寸与模型训练尺寸不一致。
- 灰度/RGB 通道不一致。
- 模型本身精度有限。
- 课堂远景人脸太小。

解决：

1. 裁剪时增加 margin。
2. 检查模型输入尺寸，例如 48x48。
3. 检查模型训练时使用的是灰度图还是 RGB 图。
4. 在报告中说明远景、遮挡、侧脸会影响准确率。
5. 如果基础模型效果差，可以尝试 DeepFace 或更好的预训练模型。

---

### 21.3 Streamlit 重复加载模型很慢

使用缓存：

```python
@st.cache_resource
def load_recognizer():
    return ExpressionRecognizer(backend="deepface")
```

---

### 21.4 CSV 中文乱码

保存时使用：

```python
encoding="utf-8-sig"
```

读取时同样使用：

```python
pd.read_csv("records.csv", encoding="utf-8-sig")
```

---

### 21.5 视频分析速度太慢

解决：

1. 不要逐帧分析，改为 1 秒 1 帧。
2. 降低视频分辨率。
3. 只在抽样帧上识别表情。
4. 使用 OpenCV Haar 而不是更重的人脸检测模型。
5. 如果使用 DeepFace，视频分析会比较慢，建议只分析短视频或减少采样频率。
